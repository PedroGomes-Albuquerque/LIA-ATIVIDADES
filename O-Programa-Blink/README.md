# O Programa Blink

O programa blink é um projeto bem simple onde você fará um led piscar, sendo o led built in do arduino, e também o led conectado ao pino 13 de seu arduino.

## Montagem do Projeto

* Peças Necessárias
  * 01 Resistor 220 ohm
  * 01 LED
  * Protobord
  * Fios para conexão
